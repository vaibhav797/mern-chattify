## Used Libraries
- `react-js`
- `tailwind css`
- `node-js`
- `express-js`
- `mongoDB`
- `socket.io`
- `jwt`



## Features

- Realtime Private Chat.
- Create or login to your account to start a conversation.
- Can search a user to start new conversation.
- Shows online status of users.
